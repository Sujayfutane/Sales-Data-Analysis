# Sales Data Analysis Project

This project is a **Python-based sales data analysis pipeline**. It demonstrates how to clean, process, and analyze sales data, and generate insights such as revenue per product and top-selling products. It also includes optional visualizations for a dashboard-like experience.

## Project Overview
This project uses Python and pandas to perform the following tasks:

1. Load and clean sales data.
2. Handle missing values and duplicates.
3. Calculate revenue and identify top-performing products.
4. Save cleaned and analyzed datasets.
5. Optionally visualize sales data for insights.

---

## Features
- ✅ Load CSV sales data
- ✅ Handle missing values (numeric and categorical)
- ✅ Remove duplicate entries
- ✅ Reset index after cleaning
- ✅ Calculate `Revenue` column
- ✅ Aggregate revenue per product
- ✅ Identify best product by revenue
- ✅ Save cleaned and analyzed data as CSV
- ✅ Optional data visualizations (bar charts, pie charts)
